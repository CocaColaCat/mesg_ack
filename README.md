# MesgAck

Wrapper around (luosimou)[http://luosimao.com]

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mesg_ack'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mesg_ack

## Usage

```ruby
 MesgAck::MesgAck.send_ack(telephone, message)
```
